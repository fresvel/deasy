# logros_copdf.js
## Descripción
Este archivo contiene el controlador para manejar logros (logros) en formato PDF para el coordinador.
## Funciones
### logros_copdf
- **Descripción**: Devuelve la URL del PDF generado.
- **Parámetros**: 
  - `req`: Objeto de solicitud (request).
  - `res`: Objeto de respuesta (response).
- **Retorna**: Respuesta JSON con la URL del PDF.
