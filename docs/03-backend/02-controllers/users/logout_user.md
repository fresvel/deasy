# logout_user.js
## Descripción
Este archivo contiene el controlador para cerrar la sesión de usuarios, limpiando la cookie del refreshToken.
## Funciones
### logoutUser
- **Descripción**: Cierra la sesión del usuario limpiando la cookie del refreshToken.
- **Parámetros**: `req`, `res`
- **Retorna**: 
  - Éxito: JSON con mensaje de éxito.
  - Error: Respuesta de error 500.
## Dependencias
Ninguna.
